## atom使用

``` cmd-\ ```  显示或隐藏目录树

``` ctrl-0 ``` 焦点移到目录树

``` fn-F2(选择tree后) ``` 修改文件/文件夹名称

``` ctrl-shift-M ``` Markdown预览，这非常方便俺这种写手

``` ctrl-alt-b `` 安装atom-beautify后可使用，格式化代码

``` ctrl-` ``` 安装terminal-panel后可使用，调起CLI

``` ctrl-shift-U ``` 调出切换编码选项

``` cmd-t或cmd-p ``` 查找文件

``` alt-cmd-[ ``` 折叠

``` alt-cmd-] ``` 展开

``` alt-cmd-shift-{ ``` 折叠全部

``` alt-cmd-shift-} ``` 展开全部

``` ctrl-m ``` 相应括号之间，html tag之间等跳转

``` alt-shift-S ``` 查看当前可用代码片段

```  cmd+shift+f ``` 全局查找
